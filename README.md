# vim-soundcloud

Soundcloud player for Vim

![](http://go-gyazo.appspot.com/63314f2b3a439476.png)

## Usage

```
:Soundcloud
```

or

```
:Soundcloud [genre]
```

## Requirements

[CtrlP](https://github.com/ctrlpvim/ctrlp.vim)

[webapi-vim](https://github.com/mattn/webapi-vim)
